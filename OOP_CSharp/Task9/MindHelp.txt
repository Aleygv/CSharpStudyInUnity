--Для скрипта Program
1) Приветственное сообщение арены колизея
2) Пользователь делает выбор 2-х из списка бойцов (выберете первого, выберете второго)
3) Происходит бой между бойцами
4) Информация о бое выводится в консоль
5) После боя выводится сообщение о победителе (или ничьей)
6) Пользователь может выбрать новых бойцов
7) Есть возможность выйти из программы
8) Структура меню:
    Приветствие
    Посмотреть бой
    Выход
9) Как проходит бой - цикл while(warrior1.IsAlive() || warrior2.IsAlive())
    
--Для скрипта Боец
1) Можно посмотреть его характеристики:
    name
    damage
    protection
    health
2) Должна быть уникальная способность у бойца
3) Методы для бойца:
    ShowStats()
    TakeDamage()
    DealDamage()
   
--Для скрипта LuckyGuy
1) Метод атаки через random с шансом на Х2 уроном

--Для скрипта Forecaster
1) Метод атаки считать (int count) и каждый 3 удар Х2 урон

--Для скрипта Vampire
1) В метод TakeDamage() засунуть рандомное значение, которое увеличивается
    с 0 до 1. По достижении значения активируется метод лечения
2) Метод ActivateTreatment() восстанавливает здоровье на 50%

--Для скрипта Wizard
1) Метод атаки через заклинание Огненный шар
2) Счетчик, который уменьшается с каждой атакой (мана)
3) По истечении маны атакует в ближнем бою, урон меньше

--Для скрипта Trickster
1) Метод TakeDamage() уменьшает здоровье только если не уклоняется. Шанс уклонения 20% (хотя как от фаербола?)
